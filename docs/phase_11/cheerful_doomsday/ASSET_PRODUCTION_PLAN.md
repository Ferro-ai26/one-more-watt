# Cheerful Electrical Doomsday — Modular Production Plan

Status: Approved Phase 11 production strategy/estimate. Phase ownership still governs implementation.

Machine-readable inventory: `asset_production_inventory.csv`.

## Architecture

- Five scale kits share four depth bands, material swatches, lighting masks, silhouette templates, and authored cable-path resources.
- WATT uses one persistent core rig plus era shells/distributed display sockets; the face, scratches, status mark, and input relic do not change identity.
- Infrastructure uses representative `single/cluster/bank/facility` states rather than one sprite per owned unit.
- Three reusable animation families cover operator authorize/connect/activate, overload/blackout/reboot, and report/scale reveal.
- Deterministic milestone snapshots drive before/after cards, reports, and capture-ready marketing states.
- Essential UI remains Godot-native. Environment images are layered sources, never full-screen baked UI.

## Performance targets

- Current-era opaque environment bands: ≤32 MiB estimated uncompressed at 720 × 1280.
- WATT rig, display sockets, cable/light masks, and icon atlases: ≤12 MiB additional.
- Keep only current era and transition-required next-scale layers resident.
- Author masters at 2× target; inspect 320, 360, 393, 480, and 720 logical widths.
- At most three simultaneous semantic motions; no free cable physics or particle fog.

## Risks

| Risk | Mitigation |
| --- | --- |
| Sixteen unrelated paintings | Five scale kits, shared materials, persistent core, repeatable silhouettes and light logic. |
| Takeover stays a text joke | Required displaced-life detail and consequence state in every milestone. |
| Operator feels passive or coerced | Payoff-first requests, explicit authorization handshake, and meaningful existing grid choices; no new permission currency. |
| Tone becomes bleak | No victims/gore; focus on absurd repurposing and calm contradiction. |
| WATT becomes evil or monstrous | Lock two-eye core geometry and behavior; additions surround rather than mutate it. |
| Cosmic views become neon noise | Physical frames, cable routes, service access, restrained cyan, large silhouette budgets. |
| Art cost grows per purchase | Four milestone states and representative clusters. |
| Generated concept drift | Manual reconstruction, style sheets, rig masters, contextual review, and rights gate before production. |
