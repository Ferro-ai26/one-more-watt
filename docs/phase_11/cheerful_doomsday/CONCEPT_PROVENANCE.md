# Cheerful Electrical Doomsday — Concept Provenance

Status: Approved direction evidence; not production-ready.

## Raster backgrounds

Five portrait environmental backgrounds were generated with the built-in OpenAI image-generation tool on 2026-07-20. No external reference image was supplied. Prompts were authored specifically for ONE MORE WATT from the user's brief and required original unbranded retro-industrial environments, two-eye cyan WATT identity, no gore, no evil-AI treatment, and no baked readable text.

Exact selected prompts: `IMAGE_GENERATION_PROMPTS.md`.

| File | Prompt focus | Contextual review |
| --- | --- | --- |
| `backgrounds/era01_workshop_generated.png` | warm innocent desk workshop | Pass as environment study; WATT is more upright/cleanly symmetrical than final rig should be. |
| `backgrounds/era05_neighborhood_generated.png` | dim neighborhood around bright cyan substation | Pass; strongest first-suspicion contrast. Main face is infrastructure-scale and requires manual identity cleanup. |
| `backgrounds/era06_city_generated.png` | city district consumed by data/cooling campus | Pass; clearly communicates Era 6 scale and protected original core. Avoid reading cooling towers as a specific real facility in production. |
| `backgrounds/era11_planet_generated.png` | Earth as circuit/cooling system | Pass for direction; environmental engineering is intentionally impossible and must be simplified at phone scale. |
| `backgrounds/era13_solar_generated.png` | Dyson/planetary infrastructure and persistent core | Pass after one discarded iteration added an unrequested mouth. Selected version keeps two-eye displays without mouths. Planet functions remain conceptual, not scientific diagrams. |

The discarded solar iteration is not copied into the repository. Generated background rights/commercial-release status remains **unresolved/internal review only** until publisher/legal policy is recorded. None may ship merely because this package is approved.

## Controlled overlays and boards

All SVG UI overlays, storyboard, WATT evolution sheet, and report are repository-native markup created for this project. They use no external images other than the five generated backgrounds named above. Dynamic/essential text is demonstrated as vector text for review but must be rendered live by Godot in production.

PNG exports are mechanical renders from the SVG sources. `approval_board.png` is a mechanical composite of the rendered concepts. No font binary is embedded or integrated; renders use available fallback sans/mono faces. Atkinson licensing/source/hash research remains in `../CONCEPT_ASSET_RECORD.md`.

## Approval and cleanup boundary

`DEC-028` approves composition, tone, material/light rules, WATT continuity, and production strategy—not literal generated geometry. Production requires manual reconstruction or controlled layered repainting, WATT rig consistency, removal of accidental pseudo-labels, accessibility validation, provenance update, licensing approval, and in-context device review.
